## 📌 Overview

Tokened addresses a real problem in schools: low student attendance and disengagement at events. The platform tracks student participation and awards digital tokens as rewards, creating a gamification loop that incentivizes consistent attendance.

Teachers manage events and verify attendance through a secure code system. Students accumulate tokens redeemable for rewards, creating measurable engagement improvement over time.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🎟️ **Token Rewards** | Students earn tokens for attending and participating in events |
| 🔐 **Teacher Verification** | Secure teacher confirmation code system prevents attendance fraud |
| 📊 **Attendance Tracking** | Persistent MongoDB backend logs all student milestones and check-ins |
| 🏆 **Leaderboard** | Gamified ranking system to encourage healthy competition |
| 📧 **Email Integration** | Automated email notifications for account management |
| 🖥️ **Cross-Platform** | Runs on Windows 10/11, Linux, and macOS |

## 🚀 Getting Started

### Prerequisites
- Windows 10/11, Linux, or macOS
- Stable internet connection
- 25 MB disk space, 2 GB RAM

### Installation

```bash
# 1. Download and extract the ZIP
# 2. Ensure the assets/ folder is in the same directory as Tokened.exe
# 3. Run the executable
Tokened.exe
```

> **Note:** For demo purposes, the teacher confirmation code is `0000`. Use a non-sensitive password for testing.
